```java
class Solution {
    public int majorityElement(int[] nums) {
        int n = nums.length;
        int a = n/2;
        int b = 0;
        int i = 0;
        int j = 0;
        for(j = n; j > a; j--){
            for(i = 0; i < j-1; i++){
                if(nums[i]>nums[i+1]){
                    b = nums[i+1];
                    nums[i+1] = nums[i];
                    nums[i] = b;
                }
            }
        }
        return nums[a];
    }
}
```
**注意事项**：
1. 注意交换元素，不要只给一个赋值
2. 注意for循环里的条件判断
3. 注意大小号不要搞错
