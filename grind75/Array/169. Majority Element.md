## 回答

```java
class Solution {
    public int majorityElement(int[] nums) {
        int n = nums.length;
        int a = n/2;
        int b = 0;
        int i = 0;
        int j = 0;
        for(j = n; j > a; j--){
            for(i = 0; i < j-1; i++){
                if(nums[i]>nums[i+1]){
                    b = nums[i+1];
                    nums[i+1] = nums[i];
                    nums[i] = b;
                }
            }
        }
        return nums[a];
    }
}
```
**注意事项**：
1. 注意交换元素，不要只给一个赋值
2. 注意for循环里的条件判断
3. 注意大小号不要搞错

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        res, count = 0, 0
        
        for n in nums:
            if count == 0:
                res = n
            count += (1 if n == res else -1)
        return res
```
**考点**：
1. 利用majority数的数量大于n/2的条件
2. 采用 0(1) 的额外空间复杂度
