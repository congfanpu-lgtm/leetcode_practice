``` java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int i = 0;
        int j = 1;
        Loop:
        for(i = 0; i <= nums.length-2; i++){
            for(j = i + 1; j <= nums.length-1; j++){
                if( nums[i] + nums[j] == target){
                break Loop;
                }
            }
        }
        return new int[]{i,j};
    }
}
```
==**注意事项**：==
1. 注意for循环中的 < 和 <= 
2. 运用Loop跳出多个循环
3. 注意查看方法需要返回的类型
