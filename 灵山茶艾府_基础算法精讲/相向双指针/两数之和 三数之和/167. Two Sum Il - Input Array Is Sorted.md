```python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        l, r = 0, len(numbers)-1

        while  numbers[l] + numbers[r] > target:
            r -= 1
        while numbers[l] + numbers[r] < target:
            l += 1
        while numbers[l] + numbers[r] == target:
            return [l + 1, r + 1]
```

```python

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        l, r = 0, len(numbers)-1

        while True:
            n = numbers[l] + numbers[r]
            if n == target:
                break
            if n < target:
                l += 1
            else:
                r -= 1
        return [l+1, r+1]
```
**思路**：与我的回答思路一样，通过两个指针往里夹，通过死循环，来实现两个指针同时往里夹
